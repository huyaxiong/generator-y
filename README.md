- yo yx:ngFty xxx (generate angular factory)
- yo yx:ngSvc xxx (generate angular service)
- yo yx:ngDrt xxx (generate angular directive)

- yo yx:gulp (add gulp configure files)
- yo yx:webpack (add webpack configure files)
